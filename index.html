Install Mods Like a Pro

Installing mods is easier than you think! Check out these steps to get rolling:

Basic Mod Installation

  1. Back up your GTA San Andreas folder—better safe than sorry!
  2. Download the mod files (usually a .zip or .rar).
  3. Unzip them using WinRAR or 7-Zip.
  4. Check the README for any special instructions.
  5. Copy the files to the right spots in your GTA SA directory.
  6. Launch the game and enjoy your new mods!

Installing CLEO Scripts

  1. Grab the CLEO library from cleo.li and install it.
  2. Download the CLEO scripts you want (.cs files).
  3. Drop the .cs files into the CLEO folder in your GTA SA directory.
  4. Check if the script needs extra files—read the docs!
  5. Start the game, and most scripts will kick in automatically.

Using Mod Loader

  1. Download Mod Loader from its official source.
  2. Unzip the files into your GTA SA directory.
  3. Create a “modloader” folder in your GTA SA directory.
  4. Make a subfolder for each mod inside the modloader folder.
  5. Drop the mod files into their subfolders.
  6. Mod Loader handles conflicts like a champ!

Troubleshooting Tips

Hit a snag? Here’s how to fix common modding issues:

  • Game crashing? Likely a mod conflict. Remove recent mods one by one to find the troublemaker.
  • Weird textures? Make sure all files are in the right folders.
  • Mod not working? Double-check you’re using game version 1.0—it’s the modding standard.
  • Laggy game? Some graphics mods are heavy. Tweak settings or upgrade your rig.
